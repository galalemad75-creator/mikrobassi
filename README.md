# 🚐 ميكروباصي — Mikrobassi

تطبيق نقل ذكي بسيارات فان 7 راكب في مصر.

## ✨ المميزات

- 🔐 تسجيل الدخول برقم الهاتف (OTP)
- 🗺️ خريطة تفاعلية مع Leaflet/OpenStreetMap
- 🚐 حجز سيارات فان (سوزوكي / شيفروليه)
- 👨‍👩‍👧‍👦 وضع العائلة
- 💰 أسعار شفافة وعادلة
- 🛡️ ميزات الأمان والطوارئ
- 📱 PWA — يعمل بدون إنترنت
- 🌐 RTL عربي بالكامل

## 📱 التوافق

✅ **Google Play** — PWA جاهز للنشر  
✅ **App Store** — PWA عبر Safari  
✅ **الويب** — يعمل على أي متصفح  

## 🛠️ التقنيات

## 🚀 النشر

### GitHub Pages
```bash
git init
git add .
git commit -m "ميكروباصي v1.0.0"
git remote add origin https://github.com/galemad75-creator/mikrobassi.git
git push -u origin main
# ثم فعّل GitHub Pages من Settings → Pages
```

### Firebase Hosting
```bash
npm install -g firebase-tools
firebase init hosting
firebase deploy
```
